# 강남펄스 알림톡(send-alimtalk) 설정 방법

✅ **Edge Function `send-alimtalk` 는 이미 배포 완료**(서울 프로젝트, v1 ACTIVE, verify_jwt OFF).
✅ **앱 연동도 이미 배포 완료**(병원설정 토글 + 카카오톡 전송 버튼).

→ 이제 **시크릿(SOLAPI_*) 등록**만 하면 실발송이 켜집니다. (아래 "시크릿 등록"으로 바로 가세요)
   함수를 다시 배포할 필요는 없습니다. (코드 수정 시에만 재배포)

순서:
1) (완료) Edge Function 배포
2) **시크릿(SOLAPI_*) 등록** ← 지금 할 일
3) 강남펄스 병원설정에서 "알림톡 자동발송 사용" 체크 + 템플릿ID 입력

Supabase 프로젝트: **서울 리전 현행 프로젝트 `utbhlvybdutepnkvboko`**
(앱이 현재 `https://utbhlvybdutepnkvboko.supabase.co` 를 사용 중 — 반드시 이 프로젝트에 배포/시크릿 등록)

---

## 방법 A. Supabase 대시보드(가장 쉬움, 설치 불필요)

### 1. 함수 배포
1. https://supabase.com/dashboard → 프로젝트 **`utbhlvybdutepnkvboko`** (서울 리전) 선택
2. 좌측 **Edge Functions** → **Deploy a new function** (또는 "Create function")
3. 함수 이름: `send-alimtalk`
4. `index.ts` 내용을 이 폴더의 `index.ts`에서 복사해 붙여넣기 → **Deploy**
5. 배포 후 함수 설정에서 **Verify JWT: OFF**(끄기) 로 설정  ※ 앱에서 publishable 키로 호출하므로 꺼야 함

### 2. 시크릿 등록
1. 좌측 **Edge Functions → (상단) Secrets** (또는 Project Settings → Edge Functions → Secrets)
2. 아래 5개 추가:
   - `SOLAPI_API_KEY`  = 솔라피 API Key
   - `SOLAPI_API_SECRET` = 솔라피 API Secret
   - `SOLAPI_PFID` = 카카오 채널 PFID
   - `SOLAPI_SENDER` = 등록한 발신번호 (예: 0212345678)
   - `SOLAPI_TEMPLATE_ID` = 승인된 알림톡 템플릿 ID

---

## 방법 B. Supabase CLI (개발자용)

```bash
# 1) 설치 & 로그인
npm i -g supabase
supabase login

# 2) 이 폴더 기준으로 함수 배포 (JWT 검증 끄기)
supabase functions deploy send-alimtalk --project-ref utbhlvybdutepnkvboko --no-verify-jwt

# 3) 시크릿 등록
supabase secrets set --project-ref utbhlvybdutepnkvboko \
  SOLAPI_API_KEY=... \
  SOLAPI_API_SECRET=... \
  SOLAPI_PFID=... \
  SOLAPI_SENDER=0212345678 \
  SOLAPI_TEMPLATE_ID=...
```
(폴더 구조: `supabase/functions/send-alimtalk/index.ts` 로 두고 배포)

## 알림톡 템플릿 (솔라피에서 등록·승인)

```
[#{병원명}] 견적 안내
#{환자명}님, 상담 견적 금액은 총 #{합계}원 입니다.
자세한 내용은 병원으로 문의해 주세요.
문의: #{전화}
```

앱이 보내는 변수 키: `#{병원명}`, `#{환자명}`, `#{합계}`, `#{전화}`
→ 템플릿의 변수명을 **정확히 이 4개와 동일**하게 맞춰야 합니다. (다르면 발송 실패)

---

## 마지막: 앱에서 켜기
강남펄스 → **병원 설정** → "알림톡 자동발송 사용" 체크 → **알림톡 템플릿 ID** 입력(선택) → 저장.
이후 상담·견적에서 **카카오톡 전송** 버튼을 누르면, 환자 연락처로 알림톡이 자동 발송됩니다.
(연락처 없음/발송 실패 시 자동으로 클립보드 복사 방식으로 전환)
